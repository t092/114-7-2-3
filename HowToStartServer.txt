### 請在本地端執行以下步驟即可開始試玩：

**第一步：啟動後端伺服器 (臨時房間主機)**
請打開終端機 (Terminal / PowerShell)，切換到專案並啟動 Server：
```bash
cd M:\OneDrive\grifon\OneDrive\AI\VibeVoding\JAPANOCCUPYTAIWAN\server
node server.js
```
*(看到 `Server listening on port 3000` 表示啟動成功)*

**第二步：啟動前端網頁 (老師與學生介面)**
請留下剛剛那個終端機不要關閉，**再開一個新的終端機**：
```bash
cd M:\OneDrive\grifon\OneDrive\AI\VibeVoding\JAPANOCCUPYTAIWAN\client
npx vite
```
 Vite 會提供一個預覽網址給您（通常是 `http://localhost:5173/`），請用瀏覽器打開它。

### 測試玩法小訣竅：
1. 您可以開一個瀏覽器視窗點擊**「老師開房控制台」**，點選建立新房間（會顯示一組 4 碼的代號）。
